# 🎵 Audio Files Needed

Place the following audio files in this folder:

| Filename        | Description                                            |
|-----------------|--------------------------------------------------------|
| `bg-music.mp3`  | Soft birthday background music (gentle, looping)       |
| `page-flip.mp3` | Subtle paper page-flip sound effect (short, ~0.5s)     |

## Where to get free sounds:

- **bg-music.mp3** → https://pixabay.com/music/search/happy%20birthday/
- **page-flip.mp3** → https://pixabay.com/sound-effects/search/page%20flip/

Or search for "royalty free birthday background music" and "paper page flip sound effect".

**Tip:** Keep `bg-music.mp3` under 5 MB for fast loading.
If no audio files are provided, the album still works perfectly — just without sound.
